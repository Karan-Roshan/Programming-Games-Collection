# Programming-Games-Collection
A curated collection of classic beginner-friendly games implemented in multiple programming languages including **C, C++, Java, Python, and JavaScript**.

This repository demonstrates how core programming logic can be applied across different languages while exploring syntax differences, structure, and implementation styles.


## Project Objective
The goal of this repository is to:
- Strengthen programming fundamentals  
- Improve logical thinking and problem-solving skills  
- Compare implementations across multiple languages  
- Practice clean and structured coding  
- Build a strong developer portfolio  


## Games Included
### Rock Paper Scissors
- C  
- JavaScript  

### Tic Tac Toe
- Java  

### Connect Four
- Java

- Python

## Key Concepts Practiced
- Conditional statements  
- Loops and iteration  
- Functions and modular programming  
- User input handling  
- Random number generation  
- Game logic implementation  
- Basic algorithm design  


## Why This Repository?
This project highlights the ability to implement the same core logic in multiple programming languages while maintaining clean structure and readability.

It serves as both a learning journey and a portfolio showcase demonstrating cross-language development skills.
